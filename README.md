# mgmt1003

A comprehensive repository for managing presentation materials and educational resources for the MGMT1003 course. This repository contains structured content, documentation, and presentation materials designed to support course delivery and learning objectives.

## Overview

This repository serves as a centralized hub for MGMT1003 course content, utilizing Quarto for document generation and presentation creation. The materials are organized to facilitate easy access and maintenance of course resources.

## Contents

- **content/**: Core educational materials and course content
- **docs/**: Generated documentation and published materials
- **source/**: Source files for presentations and materials
- **ideas/**: Development notes and conceptual materials
- **story/**: Narrative content and case studies

## Structure

The repository is organized using Quarto's publishing system, with configuration managed through `_quarto.yml`. This enables consistent formatting and easy generation of multiple output formats for presentations and documents.

## Usage

To work with the materials in this repository:

1. Ensure you have Quarto installed on your system
2. Clone the repository to your local machine
3. Navigate to the repository directory
4. Use Quarto commands to render presentations and documents as needed

## Prerequisites

- Quarto publishing system
- Basic understanding of markdown formatting
- Familiarity with MGMT1003 course structure
